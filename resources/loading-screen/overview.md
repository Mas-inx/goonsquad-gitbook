# Goonsquad Loading Screen

A full-featured FiveM loading screen with an in-game visual builder. Design custom loading screens with drag-and-drop widgets, customize backgrounds, themes, audio, and server content.

**Version:** 1.0.0 | **Price:** $14.99

{% embed url="https://www.youtube.com/watch?v=L9knbJixkAc" %}

---

## Features

- **Visual Builder** — in-game `/lsbuilder` with drag-and-drop canvas, undo/redo, and live preview
- **17 Widget Types** — server branding, progress bar, audio player, image/video, rules, changelog, tips, staff list, stats, social buttons, keybinds, queue status, overlay menus, and more
- **Backgrounds** — video, image, slideshow, gradient, or solid color
- **Audio Playlist** — background music with skip/play/pause (Space) and mute (M)
- **Multi-Profile** — save multiple config profiles to MySQL, publish one as live
- **Layout Presets** — built-in presets and user-saved presets
- **In-Game Preview** — `/lspreview` to preview the loadscreen without reconnecting
- **FiveManage Integration** — upload videos from the builder
- **Content Library** — manage rules, changelog, staff, tips, keybinds, links, custom pages
- **Queue Support** — display queue position via export
- **Press Enter to Continue** — optional gate before spawn
- **Design Variants** — 10 global aesthetics (street-noir, glass-frost, neon-signal, etc.)
