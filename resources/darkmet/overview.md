# Goonsquad DarkMet

Dark web marketplace script for FiveM that adds a full underground economy system to your RP server.

**Price:** $29.99 | [Buy on Tebex](https://goonsquad-inc.tebex.io/package/7365474)

## Features

- Laptop-based access (no commands)
- Player-driven marketplace
- DarkCoin wallet system
- Escrow & dead drops
- Bounty hunting system
- Heat & police alerts
- Raid system
- Messaging & groups
- VPN system
- Reputation & progression

## Framework Support

- QBCore
- Qbox
- ESX
- Auto-detection included

## Dependencies

- oxmysql
- qb-core / qbx\_core / es\_extended
- ox\_inventory (recommended)
