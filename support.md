# Support & Discord

## Getting Help

- Join our [Discord](https://discord.gg) for real-time support
- Open a ticket in the `#support` channel
- Check the relevant resource documentation first

## Before Asking

1. Read the resource's documentation fully
2. Check the [Changelog](changelog.md) for recent changes
3. Search Discord for similar issues

## Support Hours

Support is handled by actual developers — expect fast, technical answers.
